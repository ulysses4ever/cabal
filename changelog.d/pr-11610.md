---
synopsis: Remove build-type: Make
packages: [Cabal, Cabal-syntax, cabal-install]
prs: 11610
---

`build-type: Make` has been removed. It was an unmaintained and untested build
system that delegated to the `make` program. In practice, no packages have used
this build type; all known packages have long since been converted to
`build-type: Simple` or `build-type: Custom`. Attempting to use
`build-type: Make` in a `.cabal` file will now result in a parse error with a
descriptive message.
